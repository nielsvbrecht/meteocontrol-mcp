# Specification: Core MCP Server for MeteoControl VCOM API v2

## Goal
Build the initial version of the MCP server to provide basic monitoring, alert, and asset information from the MeteoControl VCOM API v2.

## Scope
- Implement the core MCP server structure using the Node.js SDK.
- Create tools for:
  - Fetching real-time energy and power data.
  - Retrieving and summarizing active system alerts.
  - Getting configuration and metadata for solar assets.
- Integrate Axios for secure and efficient API requests.
- Provide clear and accurate responses to user queries.

## Deliverables
- Functional MCP server compatible with MCP clients.
- Well-documented tools and API interactions.
- Test suite with high coverage for core components.
